# myrmdb 任务清单

本文档根据 `general-assessment/` 下全部题目、`README.md` 和 `doc/项目结构文档.md` 整理，用于跟踪 myrmdb 的功能实现、测试和验收。

## 总体执行顺序

1. 完成题目一：存储管理。
2. 完成题目二：查询执行。
3. 在查询执行可用后，依次扩展 BIGINT、DATETIME、唯一索引、聚合函数、ORDER BY/LIMIT、块嵌套循环连接。
4. 在基本查询和索引稳定后，实现事务控制、并发控制、故障恢复。
5. 每完成一个题目，补充对应 SQL 或单元测试，确认 `output.txt` 和客户端返回格式符合题目要求。
